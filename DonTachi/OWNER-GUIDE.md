# Don Tachi Menu - Owner Workflow

The owner uses one page only:

1. Open `owner-editor.html`
2. Edit the menu
3. Click `Publish Live`

## Daily steps

1. Open the menu editor page.
2. Change section names, prices, items, or descriptions.
3. Click `Publish Live`.
4. Enter the publish password.
5. Wait for the success message.
6. Refresh the public menu page.

## Important

- The owner does not need to edit HTML.
- The owner does not need a GitHub token.
- The public menu updates from the published menu endpoint.

## Buttons

- `Reload Published Menu`: reloads the current live menu data
- `Publish Live`: sends the updated menu live
- `Show All Sections`: shows every section at once

## If something goes wrong

1. Click `Reload Published Menu`.
2. If the issue is still live on the public site, republish from editor or redeploy worker settings.
